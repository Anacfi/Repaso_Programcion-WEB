SE CREAN LAS ACCIONES Y EL TIPO DE OBJETO

            (label) for="txtParrafo">Texto del Parrafo (/label) --- Se inserta un tipo titulo para un area de texto
            (textarea name="txtParrafo" id="txtParrafo" cols="30" rows="10") (/textarea) --- Se crea un area de texto con un nombre y un id
            (input type="button" value="Cambiar parrafo" onclick="cambiarparrafo();") --- Se crea un boton con un id y llamando a una funcion

            
DESPUES DEL BODY SE CREAN LOS SCRPITS CON LAS ACCIONES

            function cambiarparrafo(){
                var parrafo = document.getElementById('txtParrafo').value;
                document.getElementById('texto').innerHTML = parrafo;
            }
            ---
            Se crea una variable llamada parrafo el cual, busca en el documento el area de texto y
            guarda el valor que se inserto, despues se busca en el documento el area donde se coloca el texto
            y se inserta con INNERHTML la variable 'parrafo'
